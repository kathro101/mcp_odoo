## Findings

### Bug 1: `search_records()` crashes on `None` filters (Severity: HIGH)

- **Reproduction:** `search_records(odoo, schema, filters=None)`
- **Error:** `AttributeError: 'NoneType' object has no attribute 'items'`
- **Root Cause:** `filters.items()` called without None check
- **Fix:** Changed to `(filters or {}).items()`
- **File:** `src/operations/search.py`
- **Test:** `test_search_handles_none_filters` in `tests/test_qa_edge_cases.py`

### Bug 2: `preview_record()` crashes on `None` params (Severity: HIGH)

- **Reproduction:** `preview_record(schema, params=None)`
- **Error:** `TypeError: argument of type 'NoneType' is not a container or iterable`
- **Root Cause:** `f in params` iteration on None
- **Fix:** Added `params = params or {}` guard
- **File:** `src/operations/create.py`
- **Test:** `test_preview_handles_none_params` in `tests/test_qa_edge_cases.py`

### Bug 3: `OdooClient` crashes on `None`/empty model name (Severity: MEDIUM)

- **Reproduction:** `odoo_client.search_read(None, [])` or `odoo_client.search_read("", [])`
- **Error:** `ExpatError: not well-formed (invalid token)` — XML-RPC tries to send bad XML
- **Root Cause:** No input validation on `model` parameter
- **Fix:** Added `if not model: return {"status": "error", "message": "Model name is required"}` in `execute_kw`
- **File:** `src/odoo_service/odoo_client.py`

### Bug 4: `update_record()` crashes on `None` params (Severity: LOW)

- **Reproduction:** `update_record(odoo, schema, record_id, params=None)`
- **Status:** Already handled — `if not params` catches None (falsy)
- **Test:** `test_update_handles_none_params` in `tests/test_qa_edge_cases.py`
